# Security policy

Please do not report suspected security vulnerabilities in a public issue.
Use GitHub's private vulnerability reporting for this repository when it is
available. If private reporting is not enabled, contact the maintainer through
their GitHub profile with a concise reproduction and impact description.

Basicade executes BASIC programs in WebAssembly. Reports involving sandbox
escape, cross-origin isolation, unsafe asset loading, or malicious program input
are especially useful.
