# BASIC Computer Games

This directory contains a locally imported set of programs from the 1978
*BASIC Computer Games* era. Basicade registers only the programs that pass its
WASM startup smoke test; see the catalogue and test suite for the exact subset.

The original import did not include a source URL or licence record. Before any
public redistribution, replace this note with the source URL, attribution, and
licence or permission information for this copy.
